# Enable-DataStoreRoundRobin

## SYNOPSIS

## SYNTAX

```
Enable-DataStoreRoundRobin [<CommonParameters>]
```

## DESCRIPTION
Get's all datastores in vcenter and enables Datastore Round Robin on them

## EXAMPLES

### EXAMPLE 1
```
Enable-DataStoreRoundRobin
```
