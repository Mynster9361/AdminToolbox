# Get-VMFolderPaths

## SYNOPSIS

## SYNTAX

```
Get-VMFolderPaths [<CommonParameters>]
```

## DESCRIPTION
Get's all VM respective Folders in Vcenter, exports that information to an excel file, and save the excel file your download folder.

## EXAMPLES

### EXAMPLE 1
```
Get-VMFolderPaths
```
