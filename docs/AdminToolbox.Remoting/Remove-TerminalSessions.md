# Remove-TerminalSessions

## SYNOPSIS

## SYNTAX

```
Remove-TerminalSessions [<CommonParameters>]
```

## DESCRIPTION
Pulls a list of all users sessions in a Remote Desktop Server Farm and allows you to select and remove one or many at once.
