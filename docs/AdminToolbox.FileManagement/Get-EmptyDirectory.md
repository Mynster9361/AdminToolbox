# Get-EmptyDirectory

## SYNOPSIS

## SYNTAX

```
Get-EmptyDirectory [-Path] <String> [-Recurse] [<CommonParameters>]
```

## DESCRIPTION
Get a list of empty directories within a given path

## EXAMPLES

### EXAMPLE 1
```
get-emptydirectory -path c:\temp -recurse
```
